# COMS-4115-Numera

## Lexical Grammar
- Keywords = `if | then | else | while | do | end | procedure | var | begin | print`
- Identifiers = `[a-z A-Z][a-z A-Z 0-9]*`
- Operators = `+ | - | * | / | == | != | <= |>= | < | > | and | or | not`
- Numbers = `[0-9]* | [0-9]+.[0-9]*`
- LPAR = `(`, RPAR = `)`
- Separator = `;`
