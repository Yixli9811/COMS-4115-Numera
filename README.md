# COMS-4115-Numera

## Lexical Grammar
- Keywords = `if | then | else | while | do | end | procedure | var | begin | print | main | is | in`
- Identifiers = `[a-zA-Z][a-zA-Z0-9_]*`
- Operators = `== | != | <= | >= | = | + | - | * | / | % | < | > | and | or | not`
- Numbers = `Integer | Float` Integer = `0 | [1-9][0-9]*`  Float = `[0-9]+\.[0-9]* | \.[0-9]+`
- LPAR = `(`, RPAR = `)`
- Separator = `; | ,`
- Whitespace = `[\t\n\r]+`
